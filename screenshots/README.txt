Screenshots for the project report.

To generate the images automatically:
1. Open a terminal in the "chess" folder (parent of this folder).
2. Run: python generate_screenshots.py
3. This will create: 02_board.png, 04_highlights.png, 05_clocks.png, 06_promotion.png, 06_checkmate.png, 06_restart.png

To add the login screenshot (01_login.png):
1. Run the game: python chessMain.py
2. When the login window appears, take a screenshot (e.g. Win+Shift+S or Snipping Tool).
3. Save it here as: 01_login.png

Then the full report (FULL-REPORT-3.3-Iterative-Development-and-Evaluation.md or .html) will show all images.
